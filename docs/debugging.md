# Debugging

To view the debug messages, in Insomnia:

1. Open DevTools (<kbd>⌥ ⌘ I</kbd>)
1. Switch to the Console tab, and enable the `Verbose` log level
1. Optionally, filter logs by `insomnia-plugin-github-apps-helper`
